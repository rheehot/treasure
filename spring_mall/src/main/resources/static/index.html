프로젝트 개발 환경

  • 운영체제 : MacOS(Big sur)
  • 개발환경(IDE) : IntelliJ
  • JDK 버전 : JDK 11
  • 스프링 부트 버전 : 2.5.6
  • 데이터 베이스: H2 DataBase
  • 빌드 툴 : Gradle
  • 관리 툴 : git,github

프로젝트 기술 스택

  • 프론트엔드 : HTML,CSS,BootStrap,Thymeleaf
  • 백엔드 : Spring Boot
  • 데이터 베이스 : H2

프로젝트 구현 기능

  • 회원(Member) : 로그인 및 로그아웃
  • 상품(Item) : 상품 등록/ 상품 관리/상품 수정/ 상품 조회/ 상품 상세 페이지
  • 주문(Order) : 상품 주문 / 주문 내역 조회 / 주문 취소
